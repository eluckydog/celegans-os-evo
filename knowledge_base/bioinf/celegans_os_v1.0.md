# C. elegans Operating System v1.0 — 反编译架构文档

> 摘要: 将 C. elegans (秀丽隐杆线虫) 的系统架构逆向映射为完整的操作系统模型。
> 版本: 1.0 | 日期: 2026-05-16 | 所有断言均经过 NCBI 基因数据验证
> 来源: Sulston et al. (1983), White et al. (1986), WormBook, NCBI Gene

## 2. 进程模型 (Layer 1)

### 进程树主干

```
P0 (受精卵, boot)
├── AB (前部体细胞始祖, 389细胞)
│   ├── AB.a (神经前体, 194细胞)
│   └── AB.p (咽+皮下, 195细胞)
└── P1 (后部始祖, 600+细胞)
    ├── EMS (内/中胚层始祖, 100细胞)
    │   ├── E (肠道, 20细胞)
    │   └── MS (中胚层, 80细胞)
    └── P2 (生殖系始祖)
        ├── C (后部体细胞, 47细胞)
        ├── P3 (生殖系前体)
        └── D (后部肌肉, 20细胞)
```

### 进程状态

| 状态 | 含义 | 对应 |
|------|------|------|
| RUNNING | 正在分裂/分化 | 有丝分裂中 |
| WAITING | 等待信号 | G1期 / 等待Notch |
| TERMINATED | 停止分裂 | 终末分化 |
| ZOMBIE | 等待被清除 | 凋亡中 (ced-3激活) |
| SUSPENDED | 挂起 | dauer状态 |
| FROZEN | 固定不分裂 | G0期 (神经元等) |

### 关键信号
- **fork决策**: 母源因子梯度 (skn-1/pie-1/pal-1) + 细胞间信号
- **kill精确性**: 131个细胞被精确杀死, 86%在神经系
- **回收**: 邻居细胞吞噬凋亡细胞 (≈ parent reaping zombie)

## 3. IPC协议栈 (Layer 2)

| IPC模式 | 信号通路 | 发送端基因 | 接收端基因 | 可靠性 | 范围 |
|---------|---------|-----------|-----------|-------|------|
| socket (1对1) | Notch | lag-2/apx-1 | lin-12/glp-1 | 可靠(配体-受体) | 相邻细胞 |
| 广播 (1对多) | Wnt | mom-2/lin-44 | pop-1/wrm-1/Frizzled | 不可靠(梯度依赖) | 局部 |
| RPC (远程) | EGF/RTK | lin-3 | let-23 | 可靠(配体结合) | 可达~10μm |
| 全局变量 | TGF-β | daf-7 (神经分泌) | daf-1/daf-4 | 可靠(血液扩散) | 整个生物体 |
| 信号量 | Hedgehog | grd/wrt | ptc-1 | 不可靠(竞争) | 局部 |

### 标准IPC流程 (以Notch为例)

```
细胞A (发送方)                        细胞B (接收方)
┌────────────────┐                  ┌────────────────┐
│ lag-2表达      │  配体呈现在膜上   │ lin-12表达     │
│ → 配体在膜面  │ ──── 物理接触 ──→ │ → 配体结合     │
│                │                  │ → NICD切割入核 │
│                │                  │ → 转录改变     │
└────────────────┘                  └────────────────┘
```

## 4. 系统调用表 (Layer 3)

| # | syscall | 功能 | 效应基因 | 错误处理 |
|---|---------|------|---------|---------|
| 1 | fork() | 细胞分裂 | cdc-25.1, cdk-1, cyb-1 | 着丝粒错误 → 凋亡 |
| 2 | exec() | 细胞分化 | unc-86, mec-3, lin-12 等 | 未分化 → 凋亡 */
| 3 | kill() | 程序性死亡 | ced-3, ced-4, ced-9 | 清除失败 → 炎症? |
| 4 | send() | 分泌信号分子 | lag-2, mom-2, lin-3, daf-7 | 未分泌 → 相邻细胞凋亡 |
| 5 | recv() | 接收信号 | lin-12, pop-1, let-23, daf-1 | 未接收 → 默认命运 |
| 6 | bind() | 细胞贴壁 | hmr-1, hmp-1, pat-3, ina-1 | 脱落 → 凋亡 */
| 7 | migrate() | 细胞迁移 | unc-6, unc-40, unc-5 | 错位 → 凋亡 */
| 8 | read() | 环境感知 | mec-4, odr-10, tax-4 | 错误感知 → 行为异常 |
| 9 | write() | 运动/输出 | unc-54, ace-1 | 协调失败 → 瘫痪 |
| 10 | chmod() | 表观遗传 | HMT, HDAC 复合物 | 错误修饰 → 肿瘤? |

> *// 带?的为非完全确定

## 5. 调度器 (Layer 4)

### 调度架构

```
时间轴→
受精(boot) ──── 卵裂期(快速fork) ──── 器官发生(分化) ──── 孵化(L1) ──── 幼虫期 ──── 成虫
                 │                    │                      │                    │
                 ↓                    ↓                      ↓                    ↓
            时钟中断            定时器1 (lin-4)          定时器2 (let-7)        衰老
            (15min/轮)          (L1→L2)                 (L4→成虫)
```

### 调度策略

| 策略 | 机制 | 优先级 | 适用阶段 |
|------|------|--------|---------|
| FIFO | 按母源因子梯度决定 | 高 | 早期胚胎 (1-100细胞) |
| 优先级继承 | 信号受体的激活状态 | 中 | 器官发生期 |
| 定时器触发 | miRNA时钟 (lin-4/let-7) | 中 | 幼虫发育期 |
| 事件驱动 | 外部信号(dauer诱导) | 最高 | 任何阶段 |

### dauer (系统挂起)

```
食物充足 ──→ daf-7表达 → TGF-β信号 → 正常发育 (继续调度)
食物缺乏 ──→ daf-7抑制 → daf-16入核 → 休眠 (suspend all)
               ↓
          恢复(食物出现) → daf-2信号 → 退出dauer → 继续L4
```

## 6. 驱动程序 (Layer 5)

### 神经递质协议栈

```
层次          协议         递质         基因        功能
应用层       软中断       血清素       tph-1      产卵/模式切换
传输层       紧急信号     酪胺         tdc-1      逃跑反应
网络层       慢速调制     FMRF酰胺     flp-1      长效调控
总线层       数据总线     谷氨酸       eat-4      兴奋传输
驱动层       GPIO输出     乙酰胆碱     cha-1      正向运动
            GPIO反相器   GABA         unc-25     抑制/后退
状态层       状态寄存器   多巴胺       cat-2      奖励/运动
```

### 关键回路

```
触碰撤回:
  PLM(感觉-谷氨酸) → AVB/AVD(中间-谷氨酸) → VA/VB(运动-乙酰胆碱) → 肌肉(unc-54)

咽泵吸:
  ASH(感觉-谷氨酸) → RIP(中间) → M1-5(运动-乙酰胆碱) → 咽肌肉(myo-2)

产卵:
  卵→VC(感觉) → HSN(运动-血清素) → vm2/vu1(肌肉)
```

## 7. 内存管理器 (Layer 6)

### 虚拟内存保护

| 染色质状态 | 软件类比 | 关键酶 | 功能 |
|-----------|---------|-------|------|
| H3K4me3 | MAP_READ | asd-2/set-16(COMPASS) | 激活态基因, 可读 |
| H3K27me3 | mprotect(RO) | mes-2/mes-6(Polycomb) | 抑制态基因, 只读 |
| H3K9me3 | swap(冷数据) | met-2/set-25(HMT) | 异染色质, 不可访问 |

### 垃圾回收 (Ub/SUMO)

```
蛋白需要降解
    │
    ├── 多聚泛素化(K48) → 强引用计数+ → 送蛋白酶体(rpt-1/pas-4) ≈ free()
    │
    └── SUMO化(smo-1/uba-2) → 引用计数+ → 保护, 暂不解体
```

### 事务日志 (piRNA)

```
外来核酸(病毒/转座子)
    │
    ├── rde-1/rde-4 感知 → 产生siRNA
    │
    ├── 写入piRNA库(prg-1/ppw-1) ≈ WAL(预写日志)
    │
    └── 子代继承piRNA ≈ 持久化存储(重启后不丢失)
```

### 关键洞察
- 同一基因的gain/lf产生对称bug (lin-12：两个AC / 两个VU)
- 最致命bug在系统调用核心基因(fork/kill)
- 最影响体验但不致命的: 感觉神经元驱动(mec-4)
- 核心: 突变→bug 在信息处理层面就是同构的

## 9. 已知局限

1. **无多线程** — 线虫是单细胞生物到多细胞生物的过渡, 没有真正的并行
2. **无虚拟内存** — 只有染色质状态切换, 没有真正的页面置换
3. **无DMA** — 咽部运动≈DMA但缺乏真正的独立通道
4. **无用户/内核模式** — 所有细胞一个权限级别
