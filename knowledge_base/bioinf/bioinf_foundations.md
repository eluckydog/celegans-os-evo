## 1. DNA 核心概念（生物学底层结构）

### 1.1 DNA 的组成

| 概念 | 核心事实 | 数学/信息学对齐 |
|------|---------|---------------|
| **4种碱基** | A(腺嘌呤)、T(胸腺嘧啶)、C(胞嘧啶)、G(鸟嘌呤) | 4字母符号系统，碱基配对 A-T, C-G |
| **双螺旋结构** | 两条互补链反向平行缠绕，碱基对通过氢键连接 | 一维序列上的配对约束（约束满足问题） |
| **3×10⁹ 碱基对** | 人类DNA约30亿碱基对，>98%在所有人类中相同 | 信息论：约750MB原始数据，实际信息密度更高 |
| **染色体** | 23对（46条），其中22对常染色体+1对性染色体 | 43个逻辑分区（23对），每个染色体一个独立序列 |
| **基因** | 20,000-25,000个，长度为数百到200万碱基 | 序列中的特殊子段（类似代码中的函数） |
| **线粒体DNA(mtDNA)** | 细胞核外的DNA，约16,569 bp，母系遗传 | 独立的小规模序列系统 |

### 1.2 中心法则（信息流）

```
DNA → RNA → 蛋白质
```

**转录** (Transcription)：DNA→RNA，T(胸腺嘧啶)替换为U(尿嘧啶)

**翻译** (Translation)：RNA→蛋白质，每3个碱基（密码子/codon）对应1个氨基酸

| 特性 | 值 |
|------|----|
| 密码子数量 | 4³=64种 |
| 氨基酸种类 | 20种 |
| 终止密码子 | UAA, UAG, UGA（3个，不编码氨基酸） |
| 起始密码子 | AUG（也编码甲硫氨酸） |
| 遗传密码冗余度 | 多个密码子编码同一氨基酸（如亮氨酸有6个密码子） |

### 1.3 DNA 修饰（表观遗传层级）

DNA本身可以被化学修饰，其中最重要的两类：

| 修饰类型 | 位点 | 效应 |
|---------|------|------|
| **甲基化(5mC)** | CpG二核苷酸的C上 | 通常与基因沉默相关 |
| **羟甲基化(5hmC)** | 也是CpG的C上 | 与基因激活相关，去甲基化中间体 |

核心关系：**5mC ↔ 5hmC** 之间的转换是一个关键的对立/转换结构
- 甲基化：写入（DNMT酶家族），擦除：被动稀释或TET酶主动氧化
- 此二元关系可对标Ub/SUMO的竞争框架

### 1.4 常用序列格式

| 格式 | 用途 | 特性 |
|------|------|------|
| FASTA | 核酸/蛋白序列 | >号开头注释行，后接序列 |
| FASTQ | 测序原始数据 | 比FASTA多了质量分数 |
| GFF/GTF | 基因注释 | 位置、类型、属性的表格 |
| SAM/BAM | 序列比对 | 短序列到参考基因组的对齐 |
| VCF | 变异调用 | SNP/Indel等变异的记录 |

## 5. 与蛋白质修饰的衔接

之前在蛋白质修饰（Ub/SUMO）上建立的框架可以直接平移到DNA：

| 蛋白质维度 | DNA维度 | 相同之处 |
|-----------|---------|---------|
| 20种氨基酸 | 4种碱基 | 一维序列编码，都有修饰层 |
| 赖氨酸K位点 | CpG位点 | 位点特异性修饰 |
| Ub(降解) vs SUMO(调控) | 5mC(沉默) vs 5hmC(激活) | 二元对立的修饰竞争 |
| ±5窗口(11aa) | ±N窗口(碱基) | 窗口式特征提取 |
| PTM串扰 | 表观遗传串扰 | 多层修饰相互作用 |
| DeepPCT数据集 | ENCODE/Roadmap数据 | 大量标注数据 |

**关键差异**：
- DNA的token空间只有4（ATCG）而不是20，序列编码更简单
- 但DNA甲基化数据的时空维度（不同组织/发育阶段/疾病状态）更多
- DNA修饰的"写入器/擦除器/阅读器"体系更复杂
